Tovi Image Viewer

Total Image Viewer for Mac OSX!

Tovi is the image viewer for Mac OSX, to replace Preview for viewing images. Tovi automatically discover other images to generate slideshows when you open image from Finder.

Instruction

Play next/previous, move image
Scale image
Previous/Next
Rotate Left/Right
Scale up/Scale down

Supported Image Types

GIF JPG PNG SVG TIFF, and more...

GIF Animation Play

GIF animation playing is supported, also, GIFs are scalable, rotatable and flipable.

Easy Operation

Simple keyboard shortcuts: ← → ↑ ↓, for zoom, rotate, etc.

Auto Slideshows

When you open one image from Finder, Tovi will auto discover images in the same folder, then you can navigate through these image by arrow keys.
